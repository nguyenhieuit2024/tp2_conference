## Project Overview

This repository contains the **Flask backend module for Conference Management (TP2)**, a component of the scientific conference management system.  
It provides APIs for creating and managing conferences, handling topic tracks, and validating paper submission deadlines.

### Key Features
- Conference CRUD operations (Create, Read, Update, Delete)
- Track management for conference topics
- Logic validation for submission deadlines
- Fully testable using Postman (collection file included)

### Technologies
- **Language:** Python 3.10+
- **Framework:** Flask
- **Database:** SQLite (via SQLAlchemy ORM)
- **API Testing:** Postman
